### は
1. 提示主题
```
これはぺンです。
```
2. 表示强调、对比
```
小野さんは今日は休みます。
韓国へは行きました。中国へは行きませんでした。
```
>此时は可以加在 に・で・へ・から・、まで・ど后，通常用于否定

### が
1. 提示主语
```
山が高い。
``` 
用于存在句式中
```
ここは本があります。
```
2. 表示情感，能力，欲求，所有对象
```
李さんは犬が怖いです。
小野さんは料理が上手です。
```
3. 表示好恶的对象
```
あの人が好きです。
```
4. 表示铺垫，引开话题
```
すみませんが、駅はどこですか。
```

### で
1. 动作进行的场所
```
図書館で勉強します。
```
2. 提示交通工具
```
バスで家へ帰ります。
```
3. 表示方式手段
```
新聞紙で紙飛行機を作りました。
```
4. 表示原材料
```
小麦粉で餃子を作る。
```
5. 表示原因理由
```
病気で休みました。
```
6. 表示限定范围
```
日本で富士山が一番高いです。
```
7. 表示限定数量
```
３つで５００円です。
```
8. 表示并列中顿
```
私は中国人で、学生です。
```
9. 表示花费的时间
```
駅まで30分で行けます。
```

### と
1. 表示完全列举
```
机の上に本とぺンがある。
```
2. 提示引用内容 
```
先生に「さようなら」と言った。
```
3. 表示共事的对象
```
李さんと一緒に映画を見ます。
```
4. 表示互动的对象
```
李さんと喧嘩する。
```
5. 表示假定或动作之后的续起
```
```

### も
1. 表示“也”的意思
```
今日もまた雨です。
```
2. 表示全面肯定、否定（前接疑问词）
```
ここは何もない。
```

### の
1. 表示“的”的意思
```
これぺンは私のです。
```
2. 使名词化
```
本を読みむのが好きです。
```
3. 提示主语
```
川の流れ。
```

### に
1. 表示人或物存在的场所
```
箱の中にぺンがあります。
```
2. 表示具体时间
```
毎朝7時に起きます。
```
3. 表示动作指向的对象
```
李さんに電話をかけます。
```
4. 表示频率
```
1週間に1回。
```
5. 提示移动行为的目的
```
日本へ留学に行きます。
```
6. 表示目的地
```
病院に行きます。
```
7. 表示变化
```
医者になります。
```
8. 表示附着点
```
山にする。
```
9. 表示用途
```
この写真はパスポートの申請に使う。
```
10. 表示比较、比例的基准
```
3人に一人の競争率。
```
11. 表示变化的结果
```
氷が水になります。
```

### を
1. 表示动作作用的对象
```
新聞お作ります。
```
2. 提示通过的点和经过的场所
```
李さんは毎日公園を散歩します。
```
3. 表示动作、行为的目的，结果
```
お湯を熱くなる。
```
4. 表示动作的起点、分离点
```
電車を降ります。
学校を卒業する。
```

### や
1. 表示不完全列举
```
あれやこれやどれがあなたのれすか。
```

### か
1. 表示疑问
```
これは何ですか。
```
2. 表示选择
```
私は毎朝パンかお粥を食べます。
```

### へ
1. 提示移动方向
```
李さんはどこへ行きましたか。
```
2. 表示授予物品的对象
```
母への手紙。
```

### から
1. 表示起点
```
病院から帰りました。
```
2. 表示原材料
```
酒は米から作ります。
```
3. 表示动作的发起者
```
彼からをお土産をもらいました。
```
4. 表示原因
```
疲れたから、ちょっと休んだ。
```

### まで
1. 表示时间或场所的终点
```
12時まで勉強しました。
```

### より
1. 表示比较的参照物
```
今年は去年より暑い。
```
